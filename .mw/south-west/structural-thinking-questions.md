## Priority Questions for the Veritas Agent

### 🔥 Discrepancy: MMOT Bootstrap Paradox (Facet 05)

Facet 05 uses veritas' MMOT to evaluate veritas' MMOT. The tool doing the evaluation was just built and hasn't been validated. **Name this honestly in the evaluation report** — don't pretend the evaluator is trustworthy before it's been tested. First-pass evaluation is a *demonstration*, not a *verdict*.

### 🔥 Discrepancy: `.mw/` Scope — Repo vs Workspace (Facet 03)

`.mw/` was designed as a **workspace-level** convention. Veritas is a tool repo. Should veritas have its own `.mw/`? Or should veritas' evaluation artifacts land in the **ceremony workspace's** `.mw/north/` (or `.mw/west/` for action artifacts)?

**Suggestion:** If veritas creates `.mw/`, it should hold veritas' *own* introspective cycle (its development process), not its evaluations of others. Evaluations of others belong in the workspace that commissioned the evaluation.

### 💡 Implication: Network vs Local-First (Facet 01)

The MCP is a network client calling `veritas.sanctuaireagentique.com`. The `.mw/` vision is local-first. **These need to coexist.** Suggestion: CLI has `--local` mode for evaluation artifacts (reads/writes `.mw/`), network mode for model CRUD (calls API). MCP tools could support both via a `source` parameter.

### 💡 Implication: Cross-Repo Write Access (Facet 04)

Facet 04 proposes writing rispecs in 3 external repos (mia-code, medicine-wheel, coaia-narrative). The veritas agent likely doesn't have commit access everywhere. **Produce proposals as local files + create issues linking to them.** Don't assume cross-repo write access.

### 🔎 Clarification: MMOT Iteration vs Reactive Fix (Facet 05)

The second-pass "corrections" must not become reactive problem-solving. MMOT says: acknowledge → analyze → plan → document. If corrections are applied mechanically from a checklist, that's reactive. **The corrective plan should create new structural tension** (what we want to create next), not just "fix" deficiencies. The agent should be aware of this distinction.
